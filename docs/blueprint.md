# **App Name**: PoolCare Pro

## Core Features:

- Client Management: Add, update, and manage client information, including contact details and pool details.
- Pool Management: Associate pools with clients and store specific pool information (size, type, etc.).
- Product Catalog: Maintain a catalog of available products with details (name, description, cost).
- AI-Powered Product Recommendation Tool: Suggest appropriate products for a specific pool based on its parameters and last treatment, using a generative AI tool that assists the user in assessing the state of the pool.
- Scheduling System: Schedule visits on a weekly and monthly calendar, manage pending/completed visits, with drag and drop support.
- Visit Logging: Record the products used and take before/after photos during each visit.
- Payment Tracking: Track payment status (paid/pending) for each client.
- Financial Reports: Generate financial reports (product consumption, monthly revenue, completed visits).
- Offline Mode and Sync: Enable offline functionality with local data storage (IndexedDB) and automatic Firebase synchronization when online.

## Style Guidelines:

- Primary color: Medium blue (#468B97) to evoke a sense of cleanliness and reliability, appropriate for pool maintenance.
- Background color: Very light blue (#E0F4F1), subtly desaturated, for a calm and clean feel.
- Accent color: Cyan (#79C0DC) to draw attention to key actions and elements without overwhelming the user.
- Headline font: 'Space Grotesk', sans-serif, for a techy modern look. Body font: 'Inter', sans-serif, for longer blocks of text. Code Font: 'Source Code Pro', monospace, for product IDs or other codes if needed.
- Use clear and simple icons related to pools, cleaning, clients, and products.
- Design a responsive layout suitable for both web and mobile use, emphasizing a modern and minimal aesthetic.
- Incorporate subtle animations for data loading and state changes to improve user experience.
